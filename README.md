# Feel 마사지

서울 강북구에 특화된 출장마사지 예약 안내 사이트 초안입니다.

## 방향

- 브랜드명: Feel 출장마사지
- 운영 범위: 서울 강북구 집중
- 주요 권역: 미아권역, 번동권역, 수유권역, 우이권역
- 사이트 성격: 전국/광역 나열형이 아닌 강북 생활권 특화 출장마사지 안내

## 파일

- `index.html`: 메인 랜딩 페이지
- `styles.css`: 전체 디자인
- `script.js`: 모바일 메뉴와 스크롤 reveal
